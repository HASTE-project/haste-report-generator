Prints a summary of computed interestingness values to the console.

(for AZN/microscopy use case).

Contributors: Ben Blamey

### Usage

Connects to metadata-db on `localhost:27017` - you need to forward the port!

(for given `stream_id` without a 'strm_' prefix!):
```
python3 -m haste.report_generator 2018_03_05__14_35_26_from_al
```

# Usage with `watch` (experimental)
Can be used to show 'live progress' of a test run (needs a decent internet connection)
`watch` can be installed on macosx with `brew install watch`

```
watch --interval 2 "python3 -m haste.report_generator 2018_03_18__13_16_21_from_al | tail -n 50"
```
